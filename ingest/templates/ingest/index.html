About the Library

Datasets

A dataset is a stand-alone entry (e.g. an image-volume) with metadata. A dataset can contain many 2D image files that may be assembled to form a more complex two or three-dimensional volume. Datasets are defined on disk by their “root” directory in which all image files are contained (the root directory can contain recursive subdirectories with image files).

Submissions

A submission bundle is an ingestion wrapper containing one or more related datasets and metadata. Smaller bundles are recommended — all datasets must pass validation for the bundle to be accepted.
