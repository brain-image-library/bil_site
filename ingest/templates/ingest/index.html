Datasets

A dataset is a stand-alone entry (e.g. an image-volume) with metadata. A dataset can contain many 2D image files that may be assembled to form a more complex two or three-dimensional volume. Datasets are defined on disk by their “root” directory in which all the image files for the dataset are contained (the root directory can contain recursive subdirectories that contain image files).

Submissions

A submission bundle is an ingestion wrapper that contains one or more related datasets and metadata. Submissions are tagged with identifying metadata (such as the NIH project that the data is associated with) that will be propagated to each item in the submission. In general, smaller submission bundles are recommended because all datasets within a submission must pass the validation process for the submission bundle to be accepted.
